What is an Iframe?

An iframe (inline frame) is used to embed another HTML document within the current one. It creates a rectangular region where the browser displays an external document such as a web page, video, or map.

Key Iframe Attributes

Attribute Description
srcSpecifies the URL of the embedded document
nameDefines a name for the iframe, used as a target for links
width, heightSets the dimensions of the iframe
allowSpecifies permissions like access to camera/microphone
loadingControls lazy loading behavior (lazy, eager)
