The meta tag provides metadata about an HTML document ‚Äî it doesn't appear on the page but is used by browsers, search engines, and other services.

What is the Meta Tag?

The meta tag is a self-closing element placed inside the head of an HTML document.

It provides information like:

  • Document description
  • Keywords for SEO
  • Author name
  • Character set
  • Refresh behavior
  • Page redirection

Keywords

Used to define relevant keywords for search engines.

Example: meta name="keywords" content="HTML, Meta Tags, Metadata"

Document Description

Describes the content of the page for search engine results.

Example: meta name="description" content="Learning about Meta Tags"

Document Revision Date

Indicates when the document was last updated (used by some tools).

Example: meta name="revised" content="Web Dev, 2025-04-05"

Document Refreshing

Automatically refreshes the page after a certain number of seconds.

Example: meta http-equiv="refresh" content="20"

Note: Use this carefully as it can disrupt user experience.

Page Redirection

Redirects the browser to another URL after a delay (or immediately if no delay is specified).

Example: meta http-equiv="refresh" content="99; url=https://www.github.com/onyxwizard"

This will redirect to github.com after 5 seconds.

Setting Cookies (Deprecated)

Note: Using meta tags to set cookies is outdated and not supported in modern browsers.

Old syntax: meta http-equiv="cookie" content="userid=xyz; expires=Wednesday, 08-Aug-15 23:59:59 GMT;"

Use JavaScript or backend technologies instead.

Author Name

Specifies who authored the webpage ‚Äî useful for internal documentation.

Example: meta name="author" content="AK"

Character Encoding

Defines the character encoding for the HTML document.

Example (UTF-8): meta http-equiv="Content-Type" content="text/html; charset=UTF-8"

For Chinese characters: meta http-equiv="Content-Type" content="text/html; charset=Big5"
